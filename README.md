This project demonstrates full-stack web development skills using **React** for the frontend and **Node.js + Express** for the backend. The app is fully responsive and beginner-friendly.

---

## **Features**

1. **Weather Information**
   - Fetches current temperature, location, and condition from the backend API.
   - Displays information in a visually appealing card.

2. **Currency Converter**
   - Converts INR to USD and EUR.
   - Dynamic results with proper error handling.

3. **Motivational Quotes**
   - Generates random motivational quotes.
   - Can fetch new quotes on demand.

4. **SPA Navigation**
   - Smooth tab navigation between modules.
   - URL updates dynamically based on the active section.

5. **Responsive Design**
   - Works on desktop, tablet, and mobile screens.
   - Cards are centered and spaced for readability.

---

## **Tech Stack**

- **Frontend:** React, Axios, CSS
- **Backend:** Node.js, Express
- **APIs:** Weather API (OpenWeatherMap or mock), Currency Exchange API, Quotes API (or mock data)
- **Deployment:** Render / Vercel / any hosting platform
